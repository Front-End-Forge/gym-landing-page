# 🏋️ APEX Elite Fitness — Premium Gym Landing Page

A modern, high-performance gym landing page built with **React**, **Vite**, **Tailwind CSS**, and **Framer Motion**.  
Designed with a premium dark aesthetic, smooth animations, responsive layouts, and optimized performance across all devices.

---

## 🔗 Live Demo

👉 https://gym-landing-page-nine-eta.vercel.app/

---

## ✨ Features

- ⚡ Lightning-fast performance with Vite
- 🎨 Modern dark UI with neon red accents
- 📱 Fully responsive design for all devices
- 🎬 Smooth Framer Motion animations
- 🪟 Glassmorphism UI components
- 🖼️ Optimized WebP images
- 📊 Scroll progress indicator
- 🔄 Animated loading screen
- 🎠 Auto-scrolling testimonials carousel
- 🧩 Modular reusable React components
- 🧠 Clean project architecture
- 🗺️ Embedded Google Maps section
- 📈 SEO-friendly structure

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Front-End-Forge/gym-landing-page.git
```

### 2️⃣ Navigate into the Project

```bash
cd gym-landing-page
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Start Development Server

```bash
npm run dev
```

Open:

```bash
http://localhost:5173
```

---

## 📦 Production Build
